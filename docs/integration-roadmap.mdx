# Integration Roadmap: Liftoff & PocketBase Features

## Overview

This document outlines the integration roadmap for incorporating features from Liftoff (AI Mock Interview Platform) and PocketBase into TherapyTrain. The goal is to leverage the UI/UX elements from Liftoff and the backend architecture from PocketBase while maintaining our core Supabase and Zero-Knowledge infrastructure.
