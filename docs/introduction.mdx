TherapyTrain is a cutting-edge platform that combines secure communication, AI assistance, and professional tools to enhance the therapeutic process. Our platform provides:

  <Card title="AI Assistance" icon="robot">

### Secure Communication
- End-to-end encrypted sessions

- Secure file sharing
- Crisis detection and alerts

- Session scheduling and management
- Progress tracking
- Documentation assistance
- Resource recommendations
